# Commercial Readiness Notes

## What's production-ready
- Modular pipeline architecture (swappable STT/LLM/TTS)
- Structured logging with cost tracking
- Per-device rate limiting
- Configuration file system (default.json + local.json override)

## What's NOT built yet (deferred)
- Wi-Fi captive portal for first-time setup
- OTA firmware updates
- Web UI for tasks/Pomodoro
- Custom wake word training
- Multi-device / cloud sync
- Battery support
